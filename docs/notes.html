Additional Notes

While every effort has been made to make sure this ReadyTheme performs across a variety of devices and browsers, there is only so much shimming that can be done without sacrificing performance. Here is the breakdown of browsers supported by this ReadyTheme.

  • Chrome (latest 2)
  • Firefox (latest 2)
  • Safari (11+)
    Versions of Safari lower than 11 have minimal support, this theme may not work in versions lower than 10. Generally, the site will operate, however there are layout inconsistencies due to the way the browsers supports flex-box and sizing calculations.
  • Opera (latest 2)
  • Edge (latest non-Chromium)
    Internet Explorer 11 has minimal support. It will operate, however there are layout inconsistencies due to the way the browser supports flex-box and sizing calculations.

With the importance of inclusivity in website layout in mind, this ReadyTheme ships with passing grades for WCAG 2.1 Level A and Level AA as well as Section 508 standards for accessibility as tested through both Axe Developer Tools by Deque Labs and WAVE Accessibility Extension by WebAIM.
